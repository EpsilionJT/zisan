Get Start for your Pytorch Environment

JintuZheng Jan 25th 2020

This blog is written for totally beginners, don't worry you are still blank for CV and ML. I will tell you how to build an environment base on Pytorch starting in zero.

Pytorch is an open source Python ML library from Facebook Artificial Intelligence Institute (fair).

Step1: Install Python 3.6

Download Installation package in python official website https://www.python.org/downloads/

Now python has many new versions, but i suggest you to use python 3.6 as your ml platform environment, because some packages in zisan need python 3.6.X, if you use 3.7 maybe happen some unknown errors.

Step3: Install CUDA+CUDNN

Here just give the short way on Windows. Download CUDA+CUDNN on this url: link：https://pan.baidu.com/s/1JzwrxpUzhtnNj0urc-Uvxg key：5z3l

Next, you can refer to this blog for configuration https://blog.csdn.net/qq_37296487/article/details/83028394

Step2: Install Pytorch

Firstly, you can find download urls in pytorch official website. All Optional configurations depend on your platforms.

Now copy the url command, parse it on the terminal board, run then wait for download. Check if you are install pytorch sucessfully Run

import torch
import torchvision

See if there are any errors, if none, OK, Congratulations on your first success

Please see the next page to install zisan.
